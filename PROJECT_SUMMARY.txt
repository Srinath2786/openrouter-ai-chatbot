   - Real-time analysis display

4. Message.class - Conversation Model
   - Stores message role (user/assistant)
   - Stores message content
   - Part of conversation history

5. NLPAnalysis.class - Analysis Results
   - Keyword list
   - Intent classification
   - Sentiment score
   - Original text reference

HOW TO RUN:
===========

STEP 1: Set API Key (Choose One Method)
Method A - Environment Variable (Recommended):
  setx OPENROUTER_API_KEY "<YOUR_OPENROUTER_API_KEY>"
  
Method B - Dialog Prompt:
  Just run the app without setting the variable
  A dialog will ask for your API key

STEP 2: Run Application
  java Chat

STEP 3: Start Chatting!
  - Select a model from dropdown
  - Type your message
  - Press Send or Enter
  - View NLP analysis and response

TECHNICAL DETAILS:
===================

Java Version: 11+
Required Libraries: None (uses only Java standard library)
Encoding: UTF-8
Network: HTTPS (secure)
Threading: Async API calls (non-blocking UI)
Swing Components: JFrame, JPanel, JTextArea, JComboBox, JButton
API: OpenRouter REST API v1

API Endpoints:
  - Main: https://openrouter.ai/api/v1/chat/completions
  - Keys: https://openrouter.ai/keys

Authentication:
  - Bearer token in Authorization header
  - X-Title and HTTP-Referer headers included

Architecture:
  - Main class: Chat extends JFrame
  - Separate API client class for modularity
  - Separate NLP processor class for reusability
  - Message history as List for scalability

TROUBLESHOOTING:
================
Issue: java.lang.ClassNotFoundException: Chat
Solution: Make sure you're in the correct directory (e:\SD\LAB 8)

Issue: API Key not found
Solution: Set OPENROUTER_API_KEY environment variable or enter it in the dialog

Issue: Connection timeout
Solution: Check internet connection, verify API key validity

Issue: IllegalArgumentException in HTTP connection
Solution: Ensure API key format is correct, check URL encoding

For more help: https://openrouter.ai/docs
