QUICK START GUIDE - OpenRouter Chatbot
======================================

OPTION 1: Run with Environment Variable Set
--------------------------------------------
1. Open Command Prompt
2. Run these commands:
   
   setx OPENROUTER_API_KEY "sk-or-v1-..."
   
   (Replace "sk-or-v1-..." with your actual OpenRouter API key)

3. Close Command Prompt and reopen it

4. Navigate to the project:
   cd e:\SD\LAB 8

5. Compile (if not already done):
   javac Chat.java

6. Run the application:
   java Chat

7. The chatbot window opens - ready to use!


OPTION 2: Run with Dialog Prompt
---------------------------------
1. Open Command Prompt
2. Navigate to project:
   cd e:\SD\LAB 8

3. Run the application:
   java Chat

4. A dialog box appears asking for your API key
5. Paste your OpenRouter API key
6. Click OK
7. Chatbot is ready!

TROUBLESHOOTING:
================

Q: Error when running "java Chat"
A: Make sure you've compiled first: javac Chat.java

Q: "API Key not configured" error
A: Either:
  - Set OPENROUTER_API_KEY environment variable
  - Or paste it in the dialog when app starts

Q: Blank dialog when starting
A: Enter your OpenRouter API key and click OK

Q: Connection refused errors
A: Check your internet connection and API key validity

Q: Compilation errors
A: Ensure Java 11+ is installed: java -version
