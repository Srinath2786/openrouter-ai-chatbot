   - Application sends to OpenRouter API
   - UI remains responsive

7. WATCH NLP ANALYSIS
   Real-time keyword extraction and intent detection displayed

8. RECEIVE RESPONSE
   AI assistant responds after 2-5 seconds
   Full response appears in chat area

9. CONTINUE CONVERSATION
   - Conversation history is maintained
   - Models remember context
   - More natural, coherent responses
   - Switch models anytime
